- No third-party dependency

### Files Created:

### Next Steps for User:
1. Install gws: `brew install googleworkspace-cli`
2. Setup: `gws auth setup`
3. Login: `gws auth login -s gmail`
4. Test: `gws gmail +triage`
5. Run check: `python3 scripts/gmail-check.py`

### Useful Commands:
```bash
# Triage unread emails
gws gmail +triage

# Watch for new emails
python3 scripts/gmail-watch.py

# Check accounts
python3 scripts/gmail-check.py --recent
```
