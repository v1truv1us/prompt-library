## Gmail Management System - Complete Setup

Built comprehensive Gmail email management system using Google Workspace CLI (gws):

### Files Created
1. **scripts/gmail-check.py** - Check unread emails (uses `gws gmail +triage`)
2. **scripts/gmail-manage.py** - Auto-label and sort emails based on rules
3. **scripts/gmail-watch.py** - Live watch for new emails
4. **scripts/gmail-setup-labels.py** - Setup helper
5. **scripts/gmail-config.json** - Labeling rules and account config

### Usage
```bash
# Export credentials (one-time)
gws auth export --unmasked > ~/.config/gws/ACCOUNT.json

# Daily management
python3 scripts/gmail-manage.py

# Check unread
python3 scripts/gmail-check.py --recent

# Live watch
python3 scripts/gmail-watch.py
```

### Key Learnings
- Testing mode limits to ~25 OAuth scopes
- Use `gws auth login -s gmail` (not all services)
- Must add yourself as Test User in OAuth consent screen
- One gcloud project serves multiple Gmail accounts (same OAuth client)
